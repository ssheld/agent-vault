# Handoffs

Use this folder for handoff notes when work changes hands or pauses with meaningful unfinished work.

Read order:
- First read the handoff note referenced by the latest `agent-vault/context-log.md` entry.
- If none is referenced, read the most recent note in this folder.
- Each note should include current state, next steps, relevant files, validation remaining, and a suggested next prompt.

Suggested filename:
- `YYYY-MM-DD-HHMM-<from>-to-<to>-<topic>.md`
