# Daily Notes

Use one file per local day for short-lived working context.

Daily notes are supplemental. Durable decisions, open questions, and handoffs must also be captured in `agent-vault/context-log.md`, `agent-vault/decision-log.md`, `agent-vault/open-questions.md`, `agent-vault/design-log/`, or `agent-vault/context/handoffs/` as appropriate.

Read today's note first if it exists.
Create or update `YYYY-MM-DD.md` on the first substantive work session of the day.
Keep same-day content chronological: append new work log entries or added session sections at the bottom rather than inserting newer work at the top.

Suggested filename:
- `YYYY-MM-DD.md`
